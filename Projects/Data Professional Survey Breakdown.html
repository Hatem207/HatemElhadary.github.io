Power BI Project

Data Professional
Survey Breakdown

Project Overview

This Power BI dashboard provides an interactive analysis of a global survey conducted among data professionals. It explores respondents' demographics, career paths, salaries, job satisfaction, programming language preferences, and challenges faced when entering the data industry.

  • Pivot Tables
  • Pivot Charts
  • KPIs
  • Slicers
  • DAX
  • Data Modeling
  • Data Cleaning

Business Problem

Organizations and aspiring data professionals need insights into the current data job market. Understanding salary trends, preferred programming languages, job satisfaction, and career challenges helps support workforce planning, career development, and informed decision-making.

Key Insights

Data Analysts and Data Scientists represent a large portion of survey participants.
Salary levels vary considerably across different data careers.
Python is the most preferred programming language among respondents.
Job satisfaction and work-life balance differ depending on the role and experience level.
Breaking into the data industry remains one of the biggest challenges reported by participants.
