Project Overview

This Power BI dashboard analyzes Budget vs Actual performance, helping stakeholders monitor financial performance, identify variances, and support data-driven decision-making.

  • Pivot Tables
  • Pivot Charts
  • KPIs
  • Slicers
  • Power Query
  • DAX

Business Problem

Organizations need to compare planned budgets with actual performance to identify overspending, underspending, and areas requiring attention.

Key Insights

Compare Budget vs Actual across departments.
Identify departments exceeding budget.
Monitor monthly financial performance.
Track variance trends over time.
