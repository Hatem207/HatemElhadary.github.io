Power BI Project

Hospital Analysis

Project Overview

Dashboard provides a comprehensive analysis of hospital operations and patient data. It helps healthcare stakeholders monitor key performance indicators, evaluate patient outcomes, optimize resource utilization, and support data-driven decision-making.

  • Pivot Tables
  • Pivot Charts
  • KPIs
  • Slicers
  • DAX
  • Data Modeling
  • Data Visualization

Business Problem

Hospitals generate large volumes of operational and clinical data. Decision-makers need an interactive dashboard to monitor patient admissions, treatment outcomes, departmental performance, and overall hospital efficiency.

Key Insights

Monitor patient admissions over time.
Compare department performance.
Analyze patient demographics.
Evaluate hospital resource utilization.
Identify trends that support operational improvements.
